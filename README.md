# Lexisg-frontend-intern-test

## How to run the project

1. Clone the repo:
   ```
   git clone https://github.com/RajSingh0210/Lexisg-frontend-intern-test.git
   cd Lexisg-frontend-intern-test
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the dev server:
   ```
   npm start
   ```
   or
   ```
   npm run dev
   ```
